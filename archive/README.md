# Archive

Contains earlier experimental scripts used during development. These are preserved for reference but are **not required** to reproduce the paper's results.

| File | Purpose |
|------|---------|
| `Blk.py` | Initial black hole mass exploration |
| `OkFinal.py` | Early analysis draft |
| `Overmassive.py` | Testing overmassiveness calculations |
| `Teest.py`, `Testy.py` | Debug scripts |
| `Verdict.py` | Preliminary statistical tests |
| `cluster_analysis.py` | Cluster mass correlation tests |
| `powerlaw.py` | Power-law fitting experiments |

**To reproduce the paper, use only `analysis.py` in the root directory.**
